btensor.basis.get_common_parent(basis1, basis2)[source]

Get common parent of two bases.

Parameters

  • basis1 (Union[Basis, btensor.basis._NoBasis]) – Basis 1.

  • basis2 (Union[Basis, btensor.basis._NoBasis]) – Basis 2.

Returns

Common parent of basis1 and basis2. Returns nobasis if both bases are equal nobasis.

Union[Basis, btensor.basis._NoBasis]
